# My Python Project

## Setup

1. Create and activate the virtual environment:

   ```bash
   python -m venv venv
   .\venv\Scripts\Activate.ps1 # On Windows
   source venv/bin/activate # On macOS/Linux
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
